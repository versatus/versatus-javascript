  • addOrExtendMethodStrategy(methodName, newStrategyFn, extend?): void

  • Adds a new strategy function to methodStrategies or extends an existing one.

    Parameters

    • methodName: string

      The name of the method for which the strategy is defined.

    • newStrategyFn: Function

      The new strategy function to be added or used for extending.

    • Optional extend: boolean = false

      Indicates whether the new strategy should extend an existing strategy (if any).

    Returns void

    Description

    If extend is true and a strategy exists for methodName, the existing strategy is called first, and its result is passed to newStrategyFn along with the original parameters.
